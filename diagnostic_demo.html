🚀 Key Features Implemented

  • Auto-Hover Diagnostics: Popup appears automatically when hovering over problematic code (300ms delay)
  • Enhanced Visual Icons: Clear severity indicators with appropriate colors
  • Severity-Based Navigation: Navigate errors and warnings separately
  • Toggle Control: Enable/disable auto-hover with <leader>lh
  • Non-Intrusive Design: Popups disappear when cursor moves

⌨️ Enhanced Keybindings

Diagnostic Navigation:

  • [d / ]d - Navigate any diagnostic
  • [e / ]e - Navigate errors only
  • [w / ]w - Navigate warnings only
  • <leader>ld - Show diagnostic popup manually
  • <leader>lh - Toggle auto-hover on/off
